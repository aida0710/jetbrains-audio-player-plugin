# Audio Player - JetBrains IDE プラグイン

[![JetBrains Marketplace](https://img.shields.io/jetbrains/plugin/v/30608-audio-player.svg)](https://plugins.jetbrains.com/plugin/30608-audio-player)

JetBrains IDE 上で音声ファイルを直接再生・解析できるプラグインです。

## 機能

### 再生
- 再生 / 一時停止 / 停止 / シーク / 音量調整 / ループ再生
- 音程を保ったままの再生速度変更（0.5x〜2.0x、ffmpeg `atempo`）
- 再生中の VU / ピークレベルメーター
- キーボードショートカット
  - `Space`：再生 / 一時停止
  - `←` / `→`：5 秒戻る / 進む
  - `Home`：先頭へ

### 波形・スペクトラム
- **波形**：ファイルを開くと自動生成。再生位置ライン・再生済みの塗り分け・時間ルーラーを重ね、**画像をクリック / ドラッグでシーク**。L/R 分離表示にも対応。
- **スペクトラム**：スペクトログラム画像を表示（周波数軸などの外枠を含むため、時間軸の操作は波形のときのみ有効）。
- **ズーム / 横スクロール**：表示区間を拡大・移動でき、その区間を再生成して細部を確認可能。
- **ビジュアライザの表示 / 非表示トグル**：不要なときは下半分を隠してコンパクト表示に。

### 情報・ユーティリティ
- ファイルメタデータ表示（エンコーディング、フォーマット、チャンネル数、サンプルレート、ファイルサイズ、再生時間、Title / Artist / Album などのタグ）
- 情報のクリップボードコピー
- ラウドネス測定（統合ラウドネス / LUFS、ffmpeg `ebur128`）
- 別フォーマットへの書き出し（ffmpeg）
- 波形 / スペクトラム画像の PNG 保存

### IDE 統合
- プロジェクトツリーの右クリックメニュー「Audio Player で開く」
- 音声ファイルのドラッグ & ドロップで開く
- ffmpeg / ffprobe の自動検出（macOS / Linux / Windows）と設定画面。未検出時はインストール案内を通知。

### 設定の記憶
- 音量・ループ・再生速度・表示種別（波形 / スペクトラム）・ビジュアライザ表示・L/R 分離をセッションをまたいで記憶。

## 対応フォーマット

mp3, wav, ogg, flac, aac, m4a, wma, opus, ape, aiff, aif

## UI レイアウト

```
┌────────────────────────┬────────────────────────┐
│ ファイル情報 / タグ        │ ファイル名 / 再生位置      │
│                         │ 再生 / 停止 / ループ        │
│ [情報をコピー]            │ シークスライダー            │
│ [ラウドネス測定]          │ 音量 / 速度 / レベルメーター │
│ [別形式で書き出し]         │ ビジュアライザ表示トグル     │
├────────────────────────┴────────────────────────┤
│ [Waveform] [Spectrum] [画像を保存]                  │
│ [−] [＋] [全体] [L/R分離]                            │
│ 波形 / スペクトラム（波形はクリックでシーク）            │
│ [──────────── 横スクロール ────────────]            │
└─────────────────────────────────────────────────┘
```

ビジュアライザ（下半分）はトグルで非表示にでき、その場合は上半分のみのコンパクト表示になります。

## 必要環境

- **JetBrains IDE** 2024.3 以降
- **ffmpeg** — 音声再生、波形 / スペクトラム生成、速度変更、ラウドネス測定、書き出しに必要
- **ffprobe** — メタデータ表示に必要

ffmpeg / ffprobe は PATH や一般的なインストール先から自動検出します（macOS / Linux / Windows）。見つからない場合は設定画面（Settings › Tools › Audio Player）で明示的にパスを指定できます。

### ffmpeg のインストール

```bash
# macOS
brew install ffmpeg

# Ubuntu / Debian
sudo apt install ffmpeg

# Windows
winget install ffmpeg
```

## ビルド

```bash
./gradlew buildPlugin
```

`build/distributions/` に ZIP ファイルが生成されます。

## テスト

```bash
./gradlew test
```

## リント

```bash
# チェック
./gradlew ktlintCheck

# 自動フォーマット
./gradlew ktlintFormat
```

## 開発用 IDE 起動

```bash
./gradlew runIde
```

起動した IDE で音声ファイルを開くとプラグインが動作します。
